El CSpectra2021 es una herramienta constituida en tres partes: 1. Servidor, 2. Script para el programa Reaper, 3. La presente página Web para subir audios a la base de datos (MongoDB).

Para su adecuada función esta página debe de utilizarse en conjunto con las otras dos partes, que deben descargarse del enlace de GitHub.
